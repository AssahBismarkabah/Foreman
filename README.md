# Foreman

Get work done from where decisions are made -- your team chat.

## What It Is

A conductor for engineering work that lives in your team chat.

Tell it what needs done in Slack or Discord. Foreman decides which agents to use, provisions isolated environments, tracks progress, gates destructive actions behind human approval, and reports back to the channel. Agents do the work. Foreman makes sure it is done right.
